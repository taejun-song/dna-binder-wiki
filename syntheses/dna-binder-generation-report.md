# DNA Binder Generation and Evaluation Report

End-to-end report on de novo DNA-binding protein design using [[rfdiffusion3]], comparing our generated binders against Baker Lab reference designs across 9 genome editing targets, culminating in autonomous parameter optimization via [[autoresearch-for-binder-design]].

## 1. Objective

Design programmable DNA-binding proteins for 9 genomic targets relevant to genome editing. Evaluate designs using RoseTTAFold3 (RF3) complex structure prediction with pTM and ipTM as primary metrics, following the evaluation criteria from the [[rfdiffusion3-paper]].

## 2. Evaluation Criteria (from RFD3 paper)

- **pTM >= 0.8** — global fold confidence of the designed protein
- **ipTM >= 0.7** — interface confidence for the protein-DNA complex
- **DNA-aligned RMSD < 5 A** — structural accuracy of the DNA conformation

## 3. Data Sources

### 3.1 Baker Lab Reference Designs

Source: `raw/data/RFD3 binders for Genome Editing Assay -BakerLab.csv`

- **92 pre-designed candidates** across 9 DNA targets
- Generated by Baker Lab using [[rfdiffusion3]] with target-specific DNA conformations
- Includes both initial designs and specblock re-scaffolded variants

### 3.2 Our Generated Designs

Generated using [[rfdiffusion3]] via rc-foundry on an NVIDIA H200 NVL (144GB VRAM).

- **Gen2 batch**: 952 structures across 9 targets using 1BNA as DNA template
- **Overnight batch**: 2,400+ structures generated autonomously
- **Autoresearch**: 22,400+ complexes scored across 402 experiments, 9 targets
- DNA template: PDB 1BNA (CGCGAATTCGCG, ideal B-DNA), applied to all targets via contig specification

## 4. Methodology

### 4.1 Pipeline Architecture

```
RFD3 (design generation) → CIF extraction → RF3 (protein-DNA scoring) → TSV logging
```

### 4.2 Phase 1: Baker Lab Evaluation

Scored all 92 Baker Lab candidates with RF3. 9 parallel processes on H200.

### 4.3 Phase 2: Initial Generation (Gen2 + Overnight)

1. Downloaded 1BNA.pdb from RCSB as B-DNA structural template
2. Created RFD3 JSON configs for each target with contigs mapping to 1BNA chains A (residues 1-12) and B (residues 13-24)
3. Ran RFD3 with `n_batches=25-50` per target, multiple waves
4. Scored protein-DNA complexes with RF3
5. Overnight autonomous runner cycled generation -> scoring continuously

### 4.4 Phase 3: Autoresearch Parameter Optimization

Deployed 7-8 autonomous Claude Code agents on the H200 GPU server, each managing a separate DNA target. Agents ran continuously for ~3 days (April 17-19), executing the [[autonomous-experiment-loop]] pattern:

1. Modify RFD3 parameters for a single variable
2. Generate 10-80 designs per experiment
3. Score all outputs with RF3
4. If best_ipTM improved: **keep** the parameter change. If not: **revert**.
5. Move to the next parameter in the sweep

**Parameter sweep phases per target:**
- Phase 1: Protein length (80-120, 100-140, 120-160, 140-180, 160-200, 180-220)
- Phase 2: Classifier-free guidance (CFG scale 1.0-3.0)
- Phase 3: Step scale (1.0-2.0)
- Phase 4: Noise scale (0.8-1.1) x gamma (0.0-0.6)
- Phase 5: Topology (is_non_loopy) and orientation (com/hotspots)
- Phase 6: Diffusion timesteps (100-500)
- Phase 7: Final generation at 50-80 designs with best parameters

## 5. Autoresearch Agent Session Logs

### 5.1 NFKB Agent (27 experiments, 1,952 scored)

Target: GGGGATTCCCCC | Baker Lab best: 0.753

The NFKB agent discovered early (exp002) that short proteins (80-120 aa) dramatically improve ipTM from the baseline 0.575 to **0.798** — surpassing Baker Lab immediately. All subsequent experiments (CFG, step_scale, gamma, noise, orientation, topology) failed to beat this, despite 25 more attempts. The agent's key finding: **NFKB prefers compact binders** that can wrap tightly around the palindromic GC-rich target.

| Phase | Experiments | Key finding |
|-------|------------|-------------|
| Length sweep | exp001-005 | 80-120 is optimal (ipTM 0.798 vs 0.575 baseline) |
| CFG | exp006-008 | All hurt performance (0.0-0.637) |
| Step scale | exp009-011 | No improvement (best 0.738) |
| Noise/gamma | exp012-015 | No improvement (best 0.703) |
| Topology/ori | exp016-018 | No improvement |
| Fine-tuning | exp019-027 | 80-100, 90-110, 120-160 all worse |

### 5.2 HD Agent (26 experiments, 1,872 scored)

Target: GCTTAATTAGCG | Baker Lab best: 0.803

The HD agent found that length=100-140 (exp003, ipTM 0.730) was the initial best, later refined to 110-130 (exp023, ipTM **0.733**). CFG consistently failed (0 valid outputs at scale=1.0). Step_scale=1.75-2.0 showed promise (0.718-0.723) but didn't beat baseline. The agent's gap to Baker Lab (-0.069) remained the largest among compared targets — likely due to DNA shape sensitivity of the AT-rich target.

| Phase | Experiments | Key finding |
|-------|------------|-------------|
| Length sweep | exp001-007 | 100-140 optimal (0.730), narrowed to 110-130 (0.733) |
| CFG | exp008 | Failed (0 outputs) |
| Step scale | exp009-012 | 1.75-2.0 close but no beat (0.723) |
| Noise/gamma | exp013-016 | No improvement |
| Topology/ori | exp017-018 | No improvement |
| Fine-tuning | exp019-026 | s_trans, timesteps, length refinement all worse |

### 5.3 TATA Agent (8 experiments, 496 scored)

Target: CGTATAAACG | No Baker Lab reference

TATA showed a clear trend: **longer proteins score better**. From baseline 0.501 (120-140) to 0.616 (140-180) to 0.620 (160-200) to **0.684** at 180-220. The agent was cut short (CFG failed at exp008 with 0 outputs) before exploring other parameters. The very long optimal length may reflect the need for extensive scaffold to recognize the AT-rich TATA box.

### 5.4 PNRP1 Agent (70 experiments, 5,932 scored)

Target: TGAGGAGAGGAG | Baker Lab best: 0.701

PNRP1 received the most attention with 3 overlapping agent sessions. The first agent ran the standard sweep script; a second focused on CFG optimization; a third explored longer proteins with timestep variations. Key breakthrough: **CFG=1.5 combined with very long proteins (250-300 aa)** pushed ipTM to **0.662**. The agent also found that 200 timesteps (vs default 50) improved results with CFG. Despite 70 experiments, the gap to Baker Lab (-0.038) remained.

| Phase | Experiments | Key finding |
|-------|------------|-------------|
| Length sweep | exp001-msweep_006 | 160-200 best initial (0.567) |
| CFG | agent_exp001-009 | CFG=1.5 optimal (0.654) |
| CFG + length | agent_exp010-020 | L=250-300 + CFG=1.5 (0.662) |
| Timesteps | agent_exp021-036 | 200 timesteps + CFG helps |
| Noise/gamma | agent_exp037-043 | gamma_0=0.3 explored, no improvement |

### 5.5 OCT4pt2 Agent (61 experiments, 3,416 scored)

Target: GGGCTTGCGA | Baker Lab best: 0.685

Two agent sessions ran on OCT4pt2. The first found gamma_0=0.3 (alt_exp005, ipTM **0.634**) as the best single parameter. The second explored combinations: length variations with gamma, step_scale, timesteps. No combination beat 0.634. The agents tried 61 different parameter sets — the largest sweep after CAG and PNRP1 — confirming that the generic 1BNA template is the primary bottleneck for this GC-rich target.

### 5.6 OCT4pt1 Agent (27 experiments, 1,856 scored)

Target: GGTGAAATGA | Baker Lab best: 0.646

OCT4pt1 already had a strong overnight result (ipTM 0.715). The autoresearch agent methodically refined the parameters and found that **center-of-mass orientation + noise_scale=0.9 + step_scale=1.25** at length 140-180 produced ipTM **0.700** — confirming the overnight finding. The progression was systematic: length (160-200 best) -> step_scale (1.25) -> noise (0.9) -> orientation (com) -> length refinement (140-180). This target shows the clearest evidence that parameter tuning can surpass Baker Lab's target-specific DNA approach.

| Phase | Experiments | Key finding |
|-------|------------|-------------|
| Length | agent_exp001-006 | 160-200 best (0.613) |
| Step scale | agent_exp008-011 | 1.25 optimal (0.613) |
| Noise | agent_exp012-013 | 0.9 improves (0.641) |
| Gamma | agent_exp014-015 | 0.3 and 0.6 hurt |
| Orientation | agent_exp017 | COM strategy (0.661) |
| Length refine | agent_exp019 | 140-180 with all above: **0.700** |

### 5.7 Dux4grna2 Agent (40 experiments, 1,329 scored)

Target: CAGGCCGCAGG | No Baker Lab reference

The Dux4grna2 agent found that short proteins (80-120, exp004, ipTM 0.602) worked initially, but the breakthrough came late: **very long proteins (220-260) with COM orientation** (exp039, ipTM **0.641**). The agent explored an unusually wide parameter space including s_trans, timesteps, and multiple length ranges. The improvement from 0.602 to 0.641 came from combining length and orientation — parameters that interact non-linearly.

### 5.8 CAG Agent (118 experiments, 4,867 scored)

Target: CAGCAGCAGCAG | Baker Lab best: 0.637

CAG received the most experiments (118) from 3 separate agent sessions. The key discovery: **ODE sampling (gamma=0.0) with noise_scale=0.9** (agent_exp018, ipTM **0.648**). The agents tried every parameter combination including CFG at multiple scales, s_trans variations, timesteps from 50 to 500, and is_non_loopy toggling. The CAG repeat sequence proved moderately responsive to parameter tuning, with the deterministic ODE sampler providing the edge over Baker Lab (+0.011).

### 5.9 HSTELO Agent (25 experiments, 680 scored)

Target: AGGGTTAGGGTT | Baker Lab best: 0.654

The HSTELO agent ran smaller batches (10 per experiment) due to GPU contention. It found that longer proteins (140-180) without CFG worked best (exp004->exp025 rescore: ipTM **0.600**). Despite trying CFG, step_scale, noise/gamma combinations, and topology changes, nothing closed the gap to Baker Lab. HSTELO's telomeric repeat sequence likely requires specific DNA conformations (G-quadruplex-adjacent structures) that 1BNA cannot represent.

## 7. Analysis

### 7.1 Why Baker Lab Designs Still Lead on 4 Targets

1. **Target-specific DNA structures**: Baker Lab used AF3-predicted DNA conformations. We used generic 1BNA (CGCGAATTCGCG). DNA shape varies significantly by sequence — HD's AT-rich target and HSTELO's telomeric repeat likely have distinctive groove geometries that 1BNA cannot represent.

2. **Specblock re-scaffolding**: All 6 Baker Lab passing designs are specblock variants with fixed recognition interfaces.

3. **DNA shape sensitivity**: The 4 targets where we trail (HD, PNRP1, OCT4pt2, HSTELO) include AT-rich and repeat sequences where B-DNA geometry deviates most from the generic 1BNA template.

### 7.2 Why We Beat Baker Lab on 3 Targets

1. **Exhaustive parameter search**: 402 experiments vs Baker Lab's likely smaller sweep per target.

2. **Length optimization**: Discovering that NFKB prefers very short proteins (80-120 aa) and TATA prefers very long ones (180-220 aa) — counter-intuitive findings that automated search excels at.

3. **Sampler tuning**: ODE sampling (gamma=0.0) for CAG and COM orientation for OCT4pt1 — subtle parameters that have outsized effects on specific targets.

### 7.3 Key Autoresearch Discoveries

| Discovery | Target(s) | Insight |
|-----------|-----------|---------|
| Short proteins (80-120) | NFKB | Compact binders for palindromic GC-rich targets |
| Very long proteins (180-220+) | TATA, Dux4grna2, PNRP1 | AT-rich and repeat targets need more scaffold |
| ODE sampling (gamma=0.0) | CAG | Deterministic sampling better for repeat sequences |
| COM orientation | OCT4pt1, Dux4grna2 | Center-of-mass placement improves binding geometry |
| CFG=1.5 | PNRP1 | Classifier-free guidance helps some targets |
| CFG hurts | NFKB, HD, HSTELO | Guidance can overconstrain for other targets |
| Noise_scale=0.9 | OCT4pt1, CAG | Slightly reduced noise improves sampling |

## 8. Recommendations

### 8.1 Immediate: Target-Specific DNA Structures

Generate proper DNA structures for each target sequence (using x3dna-dssr or AF3) to replace 1BNA. This is the single most impactful change for closing gaps on HD (-0.069), HSTELO (-0.054), OCT4pt2 (-0.051), and PNRP1 (-0.038).

### 8.2 Short-term: Specblock Re-scaffolding

Apply specblock to our top designs, fixing the recognition interface and regenerating the scaffold. This is how Baker Lab's passing designs were created.

### 8.3 Experimental Validation

1. **NFKB autoresearch best** (ipTM 0.798) — our highest-confidence novel design
2. **OCT4pt1_18_model_0** (ipTM 0.715) — surpasses all Baker Lab OCT4pt1 candidates
3. **CAG autoresearch best** (ipTM 0.648) — novel ODE-sampling discovery
4. Test whether compact NFKB binders and extended TATA binders transfer their recognition motifs
